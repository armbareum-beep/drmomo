# 📚 Documents Folder

이 폴더에 수의학 관련 자료를 넣어주세요!

## 지원하는 파일 형식

- **📄 PDF**: `.pdf`
- **📝 Word 문서**: `.docx`, `.doc`
- **📋 텍스트 파일**: `.txt`
- **📖 Markdown**: `.md`
- **📊 스프레드시트**: `.csv`, `.xlsx`, `.xls`
- **🌐 HTML**: `.html`, `.htm`

## 사용 방법

1. 이 폴더에 자료 파일들을 복사해주세요
2. `main.py`를 실행하면 자동으로 모든 파일을 읽어서 RAG 시스템에 로드합니다
3. 여러 개의 파일을 동시에 넣어도 됩니다!

## 예시

```
documents/
├── veterinary_guide.pdf
├── dog_diseases.docx
├── cat_nutrition.txt
├── emergency_protocols.md
└── medication_dosages.csv
```

모든 파일이 자동으로 처리되어 AI 어시스턴트가 참고할 수 있게 됩니다! 🚀
